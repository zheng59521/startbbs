#startbbs
StartBBS是一个开源的轻量社区系统，界面优雅，代码简洁，小巧高效。V2.0基于thinkphp5.15和layui重构开发，核心框架成熟，学习成本低，易于二次开发，v1.x版得到了无数站长的认可和应用。尽管重构后的v2第一版功能简单，但这是我们一个新的开始，希望一起完善，再创辉煌。

产品特点：

开源轻量高效，易于二次开发。
自带文章模块，无需要再整合其它cms。
响应式布局，适配手机，平板、电脑。

环境要求:
PHP >= 5.6.0
PDO MYSQL>=5.0
